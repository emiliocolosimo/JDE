XMLSERVICE Demo Scripts (for PHP wrapper builders)

The XMLSERVICE technology is developed by IBM and Zend, which allows you to extend PHP code to access IBM i resources. The XMLSERVICE technology is designed to work with "state full" RPG/CL programs (many open database files and other system resources), so a developer won't have to rewrite any programs just to get on the web.
-> This demo shows what happens below Zend PHP toolkit wrapper using "raw" XMLSERVICE interface (PHP + XML), which you are welcome to use with/without the new PHP toolkit.
-> Clicking on {run ibm_db2}, {run pdo_ibm}, {run odbc}, etc. will run the code with the driver and clicking on {Display Code} will display the PHP code. The scripts location is /www/zendsvr/htdocs/Samples/Toolkit/RAW.
